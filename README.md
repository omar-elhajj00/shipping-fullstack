# shipping-fullstack
Shipping is a full stack web application where the admin can register, login, create, edit, delete and show all the detailed shipments.

<img
  src="/Youtube-React.png"

## Prerequisite

node v16.17.1
composer 

## Install

cd front-end 
npm install
cd back-end
php artisan migrate

## Usage

cd front-end 
npm start
cd back-end
php artisan serve
